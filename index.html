Welcome to MogBoard!

MogBoard is a small one page site that provides real-time access to the FFXIV Market Board for any server**, any time. The page is fully open source and uses XIVAPI.com to interact with the Companion API.

If you are interested in using Market Board information on your site, please have a look at the docs here: XIVAPI Market

If you fancy messing with this page and hosting your own, you can find the source code here: Github: viion/marketboard
